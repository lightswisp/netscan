# netscan
scans local network for available devices using ping (icmp requests) and scans for all common open ports

1. chmod +x ./main.rb
2. ./main.rb (subnet mask) -P 
3. Or ruby main.rb (subnet mask) -P

-P -> this parameter is for scanning ports
